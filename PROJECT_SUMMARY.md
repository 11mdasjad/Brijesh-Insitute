## 🌐 Customization Guide

### Change Institute Name
**File:** `src/components/Navbar.jsx` (Line 17)
```jsx
<p className="font-bold text-primary-700 text-lg">Your Institute</p>
```

### Change Phone Number
**File:** `src/components/Navbar.jsx` (Line 42)
```jsx
<a href="tel:+91YOUR_NUMBER">
```

### Update Courses & Pricing
**File:** `src/pages/Courses.jsx` (Lines 10-40)
```jsx
const courses = [
  {
    title: 'Class 9 & 10',
    // Update title, subjects, duration, features, etc.
  },
];
```

### Change Primary Color
**File:** `tailwind.config.js` (Lines 14-25)
```js
colors: {
  primary: {
    500: '#YOUR_HEX_COLOR',
    600: '#YOUR_DARKER_COLOR',
  }
}
```
